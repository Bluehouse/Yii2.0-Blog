# Yii2.0-Blog

### 简介

Yii2.0 最新博客系统，入门yii框架的绝佳案例

系统基于yii2.0.15框架开发，保留框架原有的特性，适合作为二次开发的基础系统，也可以直接拿来搭建博客。目前功能模块还在不断完善中，现有功能包含一些基本模块和封装好的扩展组件（富文本编辑器、图片上传、标签组件等），系统仍在继续扩展新功能，后续会更加注重用户体验，性能也会做相应调优，请及时关注 ~
